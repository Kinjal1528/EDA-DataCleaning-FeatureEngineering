# EDA, Data Cleaning & Feature Engineering Projects

This repository contains my hands-on practice notebooks for learning **Exploratory Data Analysis (EDA)**, **Data Cleaning**, and **Feature Engineering** using Python and popular data science libraries.

The goal of this repository is to build a strong foundation in data preprocessing and analysis before moving to machine learning model building.

## 📚 Projects Included

### 1. Flight Price Prediction - Feature Engineering
**Notebook:** `Flight_Price_Prediction.ipynb`

#### Topics Covered
- Loading Excel datasets using Pandas
- Dataset inspection (`head()`, `info()`, `describe()`)
- Handling data types
- Date feature extraction
- Converting categorical information into usable features
- Feature engineering for machine learning

#### Key Feature Engineering Tasks
- Extracted:
  - Day
  - Month
  - Year
- Converted object columns into numerical formats
- Prepared data for predictive modeling
